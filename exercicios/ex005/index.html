Principais Formatações

Negrito / Destaque

Opção NÃO SEMANTICA, a TAG B - Colocando o texto entre a tag B, o texto fica em Negrito

Opção SEMANTICA, a TAG STRONG - Colocango o texto entre a tag STRONG, o texto fica em Negrito

Itálico / Ênfase

Opção NÃO SEMANTICA, a TAG I - Colocando o texto entre a tag I, o texto fica em Itálico

Opção SEMANTICA, a TAG EM - Colocando o texto entre a tag EM, o texto fica em Itálico

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK

Para alterar a cor, precisamos do CSS com a tag STYLE="BACKGROUND-COLOR:COR:", assim colocamos a cor desejada.

Para textos GRANDES e PEQUENOS, usamos as TAGS:

Para textos GRANDES, usamos a TAG: BIG - Está OBSOLETA, porém ainda funciona

Para textos PEQUENOS, usamos a TAG:SMALL

Para textos DELETADOS

Podemos marcar um texto como EXCLUIDO/DELETADO para indicar que ele deve ser lido, mas não considerado COM A TAG DEL

Para textos INSERIDOS/SUBLINHADO

Podemos marcar um texto como INSERIDO/SUBLINHADO para dar enfase e indicar que ele foi adicionado depois. COM A TAG INS

Para textos SOBRESCRITOS<, usamos a TAG SUP

Como por exemplo x20+3

Para textos SUBSCRITOS, usamos a TAG SUB

Como por exemplo H2O
